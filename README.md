# 2D Infinite World using Turtle 🐢

A dynamic Python project that simulates an infinite 2D environment using Turtle graphics. The system continuously renders new areas as the user moves, creating the illusion of an endless world.

## 🔥 Highlights
- Infinite world simulation
- Real-time rendering logic
- Smooth directional movement
- Lightweight and efficient design

## 🧠 Concepts Used
- Coordinate system handling
- Continuous rendering
- Event-based movement
- Basic game development logic

## 🛠️ Tech Stack
- Python
- Turtle Graphics
